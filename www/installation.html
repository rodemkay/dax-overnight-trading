3

EA installieren

Jetzt kommt der spannende Teil - die Installation des Expert Advisors:

1. Öffnen Sie MT5 2. Klicken Sie auf "Datei" → "Datenverzeichnis öffnen" 3. Navigieren Sie zu MQL5 → Experts 4. Kopieren Sie die Datei "daxovernight.ex5" in diesen Ordner 5. Kopieren Sie "MQL_License.mqh" nach MQL5 → Include 6. Kopieren Sie "Stochastics_RSI.ex5" nach MQL5 → Indicators 7. Starten Sie MT5 neu
Geschafft! Der EA sollte jetzt im Navigator unter "Expert Advisors" erscheinen.
4

EA aktivieren

Fast fertig! Jetzt müssen wir den EA nur noch auf den Chart ziehen:

1. Öffnen Sie einen DAX-Chart (H1 Zeitrahmen empfohlen) 2. Ziehen Sie "daxovernight" aus dem Navigator auf den Chart 3. Aktivieren Sie "Automatisches Trading erlauben" 4. Klicken Sie auf OK
Wichtig: Der EA muss auf einem beliebigen Chart laufen, handelt aber immer das in den Einstellungen konfigurierte Symbol (.DE40Cash).
5

Einstellungen optimieren

Die Standardeinstellungen sind bereits optimal, aber hier die wichtigsten Parameter:

Parameter Standardwert Beschreibung
Symbol .DE40Cash Das zu handelnde Symbol (broker-abhängig)
UseAutoLot false Automatische Lot-Berechnung aktivieren
FixedLotSize 0.05 Feste Lot-Größe wenn AutoLot aus
StartHourBuy 18 Kaufzeit Stunde (18:30)
CloseHour 10 Schließzeit Stunde (10:00)
EnableMAFilter true 200er MA Filter aktiviert
EnableStochRsiFilter true StochRSI Filter aktiviert
AutoLot-Tipp: Wenn Sie AutoLot aktivieren, passt sich die Positionsgröße automatisch an Ihr Kapital an. Beispiel: Bei 10.000€ und Einstellung 0.01 Lot je 1.000€ = 0.10 Lots

Häufige Fragen

Muss der PC die ganze Nacht laufen?

Ja, der EA muss durchgehend laufen. Alternativ können Sie einen VPS (Virtual Private Server) nutzen, der 24/7 läuft. Viele Broker bieten günstige VPS-Lösungen an.

Was ist das richtige Symbol für den DAX?

Das hängt vom Broker ab. Übliche Symbole sind: .DE40Cash, Germany40, GER40, DAX40, DE40. Schauen Sie in Ihrer Symbolliste nach oder fragen Sie den Support.

Warum wird meine Position nicht geöffnet?

Prüfen Sie: Ist heute ein Handelstag? Ist es nach 18:30 Uhr? Sind die Filter erfüllt (MA, StochRSI)? Hat der DAX heute schon mehr als 1% gewonnen? Ist die Lizenz aktiv?

Kann ich die Einstellungen ändern?

Natürlich! Aber die Standardwerte sind über Jahre optimiert. Ändern Sie erstmal nur die Lot-Größe entsprechend Ihrem Kapital. Alles andere sollten Sie erst nach ausgiebigen Tests anpassen.

Wie erkenne ich, dass der EA läuft?

Im Chart sehen Sie oben rechts die Lizenzinformationen und den EA-Namen. Ein lachendes Gesicht :-) zeigt an, dass automatisches Trading aktiv ist. Um 18:30 Uhr sollte die erste Position geöffnet werden.
